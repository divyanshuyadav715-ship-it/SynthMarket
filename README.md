## 🚀 Overview

**Adaptive Trading Protocol (ATP)** is an end-to-end Machine Learning Operations (MLOps) trading platform. It demonstrates how to train a Reinforcement Learning (RL) agent on synthetically generated financial data and securely monitor its performance in production using Drift Detection and Explainable AI (XAI).

By combining **Time-Series Generative Adversarial Networks (GANs)** for data augmentation and **Proximal Policy Optimization (PPO)** for algorithmic trading, ATP provides a highly robust, risk-aware autonomous trading engine.

## 🧠 System Architecture

The ecosystem consists of 4 primary micro-components working in tandem:

1. **The Generator (Time-Series GAN)** 🧬
   - Synthesizes infinite permutations of realistic stock market episodes.
   - Accurately models volatility clustering, black swan events, and standard random walk geometries.
2. **The Trader (RL PPO Agent)** 🤖
   - Trained over tens of thousands of timesteps inside a custom `gymnasium` environment.
   - Learns to balance the portfolio (Buy/Sell/Hold) by maximizing the Sharpe Ratio and penalizing severe drawdowns.
3. **The Observer (MLOps & XAI)** 👁️
   - Calculates **KL-Divergence** in real-time between live market streams and the GAN's training distribution.
   - Auto-triggers a "Kill Switch" if market behavior severely drifts (e.g., flash crashes).
4. **The Interface (Premium Tailwind-style UI)** 💻
   - A highly polished Streamlit Community Cloud frontend.
   - Features `plotly` area charts with Geometric Brownian Motion (GBM) overlays, smooth UI transitions, and Toast notifications.

## ⚙️ Getting Started (Local Development)

### Prerequisites
- Python 3.10+
- Docker & Docker Compose (Optional for containerized deployment)

### 1. Standard Python Setup
```bash
git clone https://github.com/divyanshuyadav715-ship-it/SynthMarket.git
cd SynthMarket
python -m venv venv
source venv/bin/activate  # On Windows: .\venv\Scripts\activate
pip install -r requirements.txt
```

### 2. Run the Full Pipeline Locally
```bash
# Start the FastAPI Backend
uvicorn api:app --reload --port 8000

# In a new terminal, start the Streamlit Dashboard
streamlit run dashboard.py
```

### 3. Retraining the Models
If you wish to modify the GAN topology or PPO hyperparameters:
```bash
# Train the Time-Series GAN
python train.py

# Train the RL Agent
python train_agent.py

# Generate Benchmark Report
python benchmark.py
```
